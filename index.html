    Projects (to date)

    Arm Chair Travel

    Utilizing both Google Maps API and the Wikipedia API, user is allowed to enter a location and taken to a map of that place, with pins dropped representing articles on Wikipedia that have matching Lat/Long coordinates. User can then visit those article pages and learn more about that area.

    YouTube AJAX Search

    Entering any word, user is presented with a selection of popular YouTube videos that match via an AJAX request and JSON data templated. Additionally, when scrolling-down, additional results are loaded automatically.

    Anglophile Quiz

    How British are you? Quiz built using jQuery, templating and object methodology presents questions and answers. User progress is tracked and presented back to the user at the end. Built for endless questions.

    Hot or Cold - Random Number

    Simple jQuery code tests a user input against a random number picked at the start of each game. UX designed by Jeyz Karthika, JavaScript by me.
